Overview

The Book Generator API allows you to generate books on various topics using different language models. This documentation provides details on how to use the API, including authentication, available endpoints, and code examples.

Authentication

To use the API, you need to include your API key in the header of each request: 

X-API-Key: YOUR_API_KEY

Endpoints

Generate Book

POST /api/generate-book

Request Body

Parameter Type Description
api string The API provider to use. Options: "openai" or "together"
model string The specific model to use (e.g., "gpt-3.5-turbo", "gpt-4", "llama-2-70b")
topic string The main topic or theme of the book
language string The language in which the book should be written
chapter_count integer The number of chapters the book should have
chapter_length string Desired length of each chapter. Options: "short", "medium", "long"

Tutorial

To generate a book, send a POST request to the /api/generate-book endpoint with the required parameters. Here's an example using JavaScript and Fetch API: 


fetch('https://tryBookAI.com/api/generate-book', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    api: 'openai',
    model: 'gpt-3.5-turbo',
    topic: 'Space Exploration',
    language: 'English',
    chapter_count: 5,
    chapter_length: 'medium'
  })
})
.then(response => response.json())
.then(data => {
  console.log(data);
  // Handle the response data
})
.catch(error => {
  console.error('Error:', error);
});
